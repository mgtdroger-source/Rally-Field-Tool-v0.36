Rally Field Tool v0.20

Field capture shell for rally GPS waypoint/control information, trip-line testing and GPS trip use. This build ports the GPS Field v13 line timing engine more faithfully, keeping Method 2 and Method P separate with Q ratings.

1. Rally Details

Enter the rally/project details for this field session.

2. Mode

Select the field task. Gather Data starts GPS and capture preparation.

Gather Data

Use this mode to capture waypoint or control reference positions in the field. GPS starts quietly in the background; New Capture opens immediately.
Select New Capture to begin.

New Capture Details

Name the location before capture so the saved point is clear later.
Capture can be saved as point only, or point plus trip line.

Capture Location

Set the phone down, press Capture Location, allow the capture settle period, then the app records 20 good GPS reads. Poor reads are rejected. If the capture is struggling with bad reads, press the capture button again to stop. The button returns to ready; press it again to start over.
